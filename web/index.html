PlexLoader

Telegram-бот для домашнего медиасервера на Synology + Plex: ищет раздачи, добавляет их в Download Station и сообщает, когда контент появился в Plex.

Установка

Быстрый старт

Запустите установщик на Synology по SSH. Полная инструкция с настройкой интеграций, переменными окружения и диагностикой находится в README.

curl -fsSL https://raw.githubusercontent.com/KiMorev/tg-torrent-bot/main/install.sh | sh

Если на NAS нет curl, в README есть вариант с wget.

Что понадобится

  • Telegram-бот из @BotFather
  • Synology Download Station
  • DSM-пользователь для PlexLoader
  • Опционально: Rutracker или Jackett
  • Опционально: Plex Media Server
  • Опционально: OpenAI API key

Открытие Plex

plex.html помогает кнопке «Смотреть в Plex» открыть приложение или Plex Web. Для перехода к конкретному фильму бот добавляет параметры key и server.
